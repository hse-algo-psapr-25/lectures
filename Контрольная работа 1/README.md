# Контрольная работа 1

Контрольная работа включает в себя задания по следующим темам:
- Динамическое программирование. Задача о распределении инвестиций между проектами.
- Решение рекуррентных соотношений:
    * Вычисление ленточного определителя;
    * Вычисление количества маршрутов в графе из трех вершин.
- Оптимальное расписание:
    * Ленточная стратегия;
    * Конвейерная задача;
    * Уровневая стратегия;
    * Лексикографическая стратегия;
    * Стратегия разделения процессоров.

Каждый вариант контрольной работы будет содержать три задачи, которые необходимо будет решить на бумаге, сфотографировать решение и черновики, объединить в один PDF файл и загрузить решение в течение 10 минут после окончания занятия.

Во время выполнения заданий контрольной работы необходимо включить камеру и микрофон.
